# Feature Graphic Quick Start Guide

## ⚡ 5-Minute Solution (Recommended)

### Use Canva - No Design Skills Required!

1. **Go to**: https://www.canva.com/
2. **Sign up**: Free account
3. **Create design**: Click "Create a design" → "Custom size"
4. **Set size**: 1024 × 500 pixels
5. **Design**:
   - Click "Background" → Choose solid color → Enter `#4ECDC4`
   - Click "Text" → Add heading → Type "SpendFlux"
   - Style text: Bold, 72pt, White color
   - Add another text → Type "Track Your Expenses Effortlessly"
   - Style: Regular, 32pt, White color
   - Click "Uploads" → Upload `assets/icons/app_icon.png`
   - Drag icon to left side, resize to ~150px
   - Position text to the right of icon
6. **Download**: Click "Share" → "Download" → PNG
7. **Save as**: `feature_graphic_1024x500.png`

**Time needed**: 5-10 minutes
**Cost**: Free
**Result**: Professional-looking graphic

### Option 2: PowerPoint/Google Slides (15 min)
✅ No account needed
✅ Familiar interface
✅ Quick and simple

**PowerPoint Steps**:
1. Open PowerPoint
2. Design → Slide Size → Custom
3. Width: 26.67 cm, Height: 13.02 cm
4. Add background color: #4ECDC4
5. Insert text boxes:
   - "SpendFlux" (Bold, 72pt, White)
   - "Track Your Expenses Effortlessly" (32pt, White)
6. Insert → Pictures → Add your app icon
7. File → Save As → PNG → "Just This Slide"

**Google Slides Steps**:
1. Go to slides.google.com
2. File → Page setup → Custom
3. Width: 26.67 cm, Height: 13.02 cm
4. Design your slide (same as PowerPoint)
5. File → Download → PNG

4. Upload: `feature_graphic_1024x500.png`
5. Preview and save

## 💡 Pro Tips

1. **Keep it simple**: Don't overcrowd
2. **Use high contrast**: White text on teal background works great
3. **Test at small size**: Zoom out to see how it looks
4. **Match your app icon**: Use same colors
5. **Get feedback**: Show to friends before uploading

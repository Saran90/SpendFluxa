# Play Store Screenshots Creation Guide

## 📱 Requirements Summary

### Phone Screenshots (Required: 2-8)
- **Format**: PNG or JPEG
- **Size**: Up to 8 MB each
- **Aspect Ratio**: 16:9 or 9:16
- **Dimensions**: 320-3,840 px per side
- **Recommended**: 1080×1920 px (9:16 portrait)

### 7-inch Tablet Screenshots (Optional: 0-8)
- **Format**: PNG or JPEG
- **Size**: Up to 8 MB each
- **Aspect Ratio**: 16:9 or 9:16
- **Dimensions**: 320-3,840 px per side
- **Recommended**: 1200×1920 px (portrait) or 1920×1200 px (landscape)

### 10-inch Tablet Screenshots (Optional: 0-8)
- **Format**: PNG or JPEG
- **Size**: Up to 8 MB each
- **Aspect Ratio**: 16:9 or 9:16
- **Dimensions**: 1,080-7,680 px per side
- **Recommended**: 1600×2560 px (portrait) or 2560×1600 px (landscape)

## 🚀 Quick Creation Methods

### Method 1: Use Figma (Recommended - Professional Results)

**Step 1: Set up Figma**
1. Go to: https://www.figma.com/
2. Create free account
3. Create new design file

**Step 2: Create Phone Frame Template**
1. Create frame: 1080 × 1920 px
2. Add phone mockup from Figma community
3. Search: "iPhone mockup" or "Android mockup"

**Step 3: Design Each Screen**
1. Create app UI inside phone frame
2. Use SpendFlux colors: #4ECDC4 (teal), #FF6B6B (coral)
3. Add realistic data and content

**Step 4: Export**
1. Select frame
2. Export → PNG → 2x scale
3. Save as: `phone_screenshot_1.png`, etc.

---

### Method 2: Use Canva (Easiest)

**Step 1: Setup**
1. Go to: https://www.canva.com/
2. Create custom size: 1080 × 1920 px

**Step 2: Add Phone Frame**
1. Search elements: "phone mockup"
2. Choose modern smartphone frame
3. Resize to fit canvas

**Step 3: Create App Screens**
1. Design app interface inside phone
2. Use rectangles, text, icons
3. Apply SpendFlux brand colors

**Step 4: Export**
1. Download → PNG
2. High quality setting

---

### Method 3: Use Screenshot Tools + Mockups

**If you have the app running:**
1. Take actual screenshots from device/emulator
2. Use online mockup generators:
   - https://mockuphone.com/
   - https://deviceshots.com/
   - https://smartmockups.com/

## 🎨 Design Guidelines

### Colors (SpendFlux Brand)
- **Primary**: #4ECDC4 (Teal)
- **Accent**: #FF6B6B (Coral/Red)
- **Background**: #F7FAFA (Light gray)
- **Text Primary**: #1A2E44 (Dark blue)
- **Text Secondary**: #6B7C93 (Gray)
- **Success**: #2D9E6B (Green)

### Typography
- **Headers**: Bold, 18-24px
- **Body**: Regular, 14-16px
- **Captions**: 12px
- **Font**: Use system fonts (SF Pro, Roboto)

### Layout Principles
1. **Consistent spacing**: 16px, 24px, 32px
2. **Card-based design**: Rounded corners (16px)
3. **Proper hierarchy**: Clear visual hierarchy
4. **Realistic data**: Use Indian currency (₹)
5. **Clean backgrounds**: Avoid clutter

## 💡 Pro Tips

### Design Tips
1. **Use real data**: Make it feel authentic
2. **Show success states**: Happy, positive scenarios
3. **Consistent branding**: Use SpendFlux colors throughout
4. **Clear hierarchy**: Important elements stand out
5. **Mobile-first**: Design for phone viewing first

### Marketing Tips
1. **Tell a story**: Screenshots should flow logically
2. **Highlight benefits**: Show value, not just features
3. **Target audience**: Use relevant scenarios (Indian context)
4. **Competitive edge**: Show unique features
5. **Call to action**: Make users want to download

### Technical Tips
1. **High resolution**: Use 2x or 3x scale
2. **Optimize file size**: Compress without quality loss
3. **Test on devices**: View on actual phones/tablets
4. **Consistent naming**: Use clear file names
5. **Backup sources**: Keep editable files

## 📤 Upload to Play Console

1. Go to: **Google Play Console**
2. Navigate: **Store presence → Main store listing**
3. Scroll to: **Screenshots** section
4. Upload phone screenshots (2-8 required)
5. Upload tablet screenshots (optional)
6. Preview how they look
7. **Save** changes

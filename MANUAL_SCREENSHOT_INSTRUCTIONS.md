## 🚀 Method 1: Browser Screenshot (Easiest - 5 minutes)

### Step 1: Open HTML Files
1. **Right-click** on `screenshot_1_home.html`
2. **Open with** → Your browser (Chrome, Edge, Firefox)
3. The phone mockup will display perfectly

### Step 2: Take Screenshot
**Option A: Browser Dev Tools (Best Quality)**
1. Press **F12** to open Developer Tools
2. Click **Device Toggle** icon (📱) or press **Ctrl+Shift+M**
3. Set device to **iPhone 12 Pro** or **Custom** 375×812
4. Right-click on page → **Capture screenshot**
5. Save as `phone_screenshot_1_home.png`

**Option B: Windows Snipping Tool**
1. Press **Windows + Shift + S**
2. Select **Rectangle** mode
3. Carefully select just the phone mockup area
4. Save as PNG

**Option C: Full Page Screenshot Extension**
1. Install browser extension like "Full Page Screen Capture"
2. Click extension icon
3. Save the screenshot

### Step 3: Repeat for Other Screens
- Open `screenshot_2_add_transaction.html` → Save as `phone_screenshot_2_add_transaction.png`
- Open `screenshot_3_analytics.html` → Save as `phone_screenshot_3_analytics.png`

## 📐 Method 3: Resize to Play Store Requirements

After capturing, you need to resize to Play Store specs:

### Current Size: 375×812 px
### Required Size: 1080×1920 px (9:16 ratio)

**Option A: Online Resizer**
1. Go to: **https://www.iloveimg.com/resize-image**
2. Upload your screenshot
3. Set to: **1080 × 1920 pixels**
4. Download resized version

**Option B: Windows Photos App**
1. Open screenshot in Photos app
2. Click **Edit & Create** → **Resize**
3. Set custom size: 1080 × 1920
4. Save

## 🚀 How to Convert to PNG (5 minutes)

### Method 1: Browser Screenshot (Recommended)
1. **Open** each HTML file in Chrome/Edge
2. **Press F12** → Click device toggle (📱)
3. **Set device** to iPhone 12 Pro or Custom 375×812
4. **Right-click** → Capture screenshot
5. **Save** as PNG

### Method 2: Online Converter
1. **Go to**: https://htmlcsstoimage.com/
2. **Upload** HTML file
3. **Set size**: 375×812 px
4. **Download** PNG

### Method 3: Resize to Play Store Size
After capturing, resize to **1080×1920 px**:
- Use: https://www.iloveimg.com/resize-image
- Or: Windows Photos app → Edit → Resize

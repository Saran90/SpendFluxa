- Larger text and icons
- More breathing room

### 10-inch Tablet (1600×2560 Portrait)

- Side-by-side layouts where appropriate
- Enhanced charts and graphs

## 📐 Responsive Breakpoints

### Phone (Portrait)
- Canvas: 1080×1920
- Content: 1000×1800 (safe area)
- Margins: 40px sides, 60px top/bottom

### 7-inch Tablet
- Canvas: 1200×1920 (portrait) or 1920×1200 (landscape)
- Content: 1080×1800 or 1800×1080
- Margins: 60px all sides

### 10-inch Tablet
- Canvas: 1600×2560 (portrait) or 2560×1600 (landscape)
- Content: 1440×2400 or 2400×1440
- Margins: 80px all sides

## 🔧 Implementation Tips

### Using Figma
1. Create components for repeated elements
2. Use auto-layout for responsive design
3. Create a style guide with colors and fonts
4. Use plugins for icons and mockups

### Using Canva
1. Create brand kit with SpendFlux colors
2. Use consistent fonts throughout
3. Save elements as templates
4. Export at highest quality

### Using Adobe XD
1. Use repeat grids for lists
2. Create symbols for common elements
3. Use responsive resize for different sizes
4. Export for screens at 2x resolution

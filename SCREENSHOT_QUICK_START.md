# Screenshot Quick Start Guide

## ⚡ 30-Minute Phone Screenshots

Create professional Play Store screenshots in 30 minutes using Canva.

---

## 🚀 Method 1: Canva (Recommended - Easiest)

### Step 1: Setup (5 minutes)
1. Go to: **https://www.canva.com/**
2. Sign up (free account)
3. Click: **Create a design** → **Custom size**
4. Enter: **1080 × 1920 pixels**
5. Click: **Create new design**

### Step 2: Add Phone Frame (5 minutes)
1. Click: **Elements** (left sidebar)
2. Search: **"phone mockup"**
3. Choose: Modern smartphone frame (iPhone or Android style)
4. Drag to canvas and resize to fit
5. Position: Center of canvas

### Step 3: Create Home Screen (10 minutes)
1. **Add background inside phone**:
   - Rectangle tool → Draw inside phone screen
   - Color: Gradient (#4ECDC4 to #38B2A9)
   - Position: Top 40% of phone screen

2. **Add header text**:
   - Text tool → "Hello, Sarah 👋"
   - Font: Bold, 24px, White
   - Position: Top left of gradient area

3. **Add balance**:
   - Text tool → "Net Balance"
   - Font: Regular, 14px, White (70% opacity)
   - Text tool → "₹45,230"
   - Font: Bold, 36px, White

4. **Add summary cards**:
   - Rectangle → White, rounded corners
   - Text → "Income ₹65,000" (Green #2D9E6B)
   - Rectangle → White, rounded corners  
   - Text → "Expenses ₹19,770" (Red #FF6B6B)

5. **Add transaction list**:
   - Rectangles → White cards
   - Text → "🍕 Lunch at Cafe -₹450"
   - Text → "⛽ Petrol -₹2,000"
   - Text → "💰 Salary +₹50,000"

### Step 4: Export (2 minutes)
1. Click: **Share** → **Download**
2. Format: **PNG**
3. Quality: **High**
4. Click: **Download**
5. Save as: `phone_screenshot_1_home.png`

### Step 5: Create More Screens (8 minutes)
Repeat for other screens using templates from SCREENSHOT_TEMPLATES.md

## ✅ 5-Screenshot Sequence

### Required Screenshots (in order):

1. **Home Screen** - Shows app overview
   - Net balance, income/expenses, recent transactions
   - Most important screenshot

2. **Add Transaction** - Key functionality
   - Transaction form, categories, amount input
   - Shows ease of use

3. **Analytics** - Advanced features
   - Pie chart, spending breakdown, trends
   - Shows value proposition

4. **Budget Management** - Financial planning
   - Budget cards, progress bars, alerts
   - Shows control features

5. **Accounts Overview** - Multi-account support
   - Account cards, balances, recent activity
   - Shows comprehensive tracking

## 🔧 Quick Tips

### Design Tips
1. **Keep it simple**: Don't overcrowd screens
2. **Use real data**: Make it feel authentic
3. **Consistent colors**: Stick to SpendFlux palette
4. **Readable text**: Ensure text is clear at small sizes
5. **Professional frames**: Use modern, clean phone mockups

### Technical Tips
1. **High resolution**: Use 2x scale for crisp images
2. **Correct aspect ratio**: 9:16 for phone (portrait)
3. **File size**: Keep under 8 MB (usually 1-3 MB is fine)
4. **PNG format**: Better quality than JPEG
5. **Consistent naming**: Use clear, descriptive file names

### Content Tips
1. **Show benefits**: Focus on what users gain
2. **Indian context**: Use ₹ currency, relevant scenarios
3. **Positive data**: Show successful financial management
4. **Clear hierarchy**: Important info stands out
5. **Story flow**: Screenshots should tell a story

## 📤 Upload Process

### Google Play Console Steps:
1. Go to: **https://play.google.com/console**
2. Select: **SpendFlux** app
3. Navigate: **Store presence** → **Main store listing**
4. Scroll to: **Screenshots** section
5. **Phone screenshots**: Upload 2-8 images (required)
6. **7-inch tablet**: Upload 0-8 images (optional)
7. **10-inch tablet**: Upload 0-8 images (optional)
8. **Preview**: Check how they look
9. **Save**: Save changes

### Upload Requirements:
- **Format**: PNG or JPEG
- **Size**: Up to 8 MB each
- **Phone**: 320-3,840 px per side, 16:9 or 9:16 ratio
- **Tablets**: Same size limits, same ratios

## 🆘 Troubleshooting

### "Image too large"
→ **Solution**: Compress using TinyPNG.com

### "Wrong aspect ratio"
→ **Solution**: Use exactly 1080×1920 for phone

### "Looks blurry"
→ **Solution**: Export at 2x resolution, use PNG format

### "Text too small"
→ **Solution**: Increase font sizes, use bold weights

### "Doesn't look professional"
→ **Solution**: Use consistent spacing, align elements properly
